하이퍼텍스트란?

‘하이퍼텍스트’(hypertext)란 링크로 연결된 정보 조직 구조이자 텍스트를 의미한다. 기술적 관점에서 하이퍼 텍스트의 정보 단위를 노드(node)라 하고, 다른 정보 단위로의 이동을 링크(link)라 부른다(Nielsen, 1995). 최근에는 하이퍼텍스트를 구성하는 정보들이 멀티미디어화되며 하이퍼미디어라고도 한다.

다음으로 하이퍼텍스트의 특징은 기존의 전통 텍스트와의 대조를 통해 알아볼 수 있다. 하이퍼텍스트의 가장 대표적인 특징인 비순차적 구조에서는 하이퍼텍스트의 링크 기능이 큰 역할을 한다. 먼저 전통 텍스트의 예시로 한글 파일에서는 페이지를 정해진 순서대로 읽어야 하지만, 하이퍼텍스트는 링크를 통해 사용자가 원하는 텍스트를 페이지 순서와 상관없이 바로 연결하여 읽을 수 있다.

이런 비순차적 구조뿐만 아니라, 하이퍼텍스트는 하이퍼미디어로써 전통 텍스트와 달리 다양한 미디어 유형을 지원한다. 단순한 텍스트에 더하여 이미지, 사운드, 동영상, 가상환경까지 멀티미디어의 형태를 띠고 있다. 이렇게 링크를 통해 정보들을 유기적으로 연결함으로써 하이퍼텍스트는 디지털 영역, 나아가 웹에 이르기까지 다양한 영역에서 활용되고 있다.

웹이란?

‘웹’(WorldWideWeb)이란 문서를 서버 컴퓨터에 올려놓고 인터넷을 통해 클라이언트 컴퓨터에서 읽거나 쉽게 주고받을 수 있는 서비스이다. 주의할 점은 웹과 인터넷이 다른 개념임을 확실히 해야 하는데, 인터넷은 컴퓨터 네트워크망 자체이고 웹은 그 인터넷 서비스 중 하나이다. 인터넷 서비스인 웹의 가장 대표적인 특징은 ‘하이퍼텍스트’ 구조이며, 이는 ‘링크로 연결된 정보 조직 구조’를 의미한다.

하이퍼텍스트가 왜 웹의 대표 특징인지 이해하기 위해서는 먼저 웹 문서의 구조를 간단히 파악할 필요가 있는데, 웹 문서(html)는 여러 웹 페이지들로 구성되며 이 웹 페이지들의 연결에 링크로 정보를 연결하는 하이퍼텍스트 개념이 적용되는 것이다. 여기서 웹 페이지들의 연결에는 ‘하이퍼링크’가 이용되며, 이 하이퍼링크를 통해 전 세계의 웹 페이지들은 단어 그대로 거미줄(web)처럼 연결된다. 이렇게 웹은 하이퍼텍스트 구조를 지님으로써, 하이퍼링크를 통한 비순차적 구조와 다양한 멀티미디어 유형을 제공하고 웹 설계자와 사용자 사이에 상호작용이 가능한 점 등 유사한 특징을 지닌다.

웹 2.0이란?

새로운 상품이 출시될 때 상품명에 붙는 것과 같이 ‘웹 2.0’ 역시 기존 웹 1.0에서 발전한 플랫폼 형태이다. 여기서 플랫폼(platform)이란 ‘어떠한 기능을 사용하기 위해 들러야 하는 공간’을 의미하며(주로 웹 브라우저의 초기페이지) 웹 서비스 또한 이 플랫폼의 역할을 한다. 기존의 웹(웹 1.0을 의미)은 하이퍼링크를 통해 다른 페이지로 연결되는 서비스를 제공했으나 웹 2.0은 웹 페이지 상에서 다양한 기능을 직접 제공하는 플랫폼 형태로 진화하였다. 예를 들자면 검색을 통해 다음 페이지로 넘어가는 방식(웹 1.0)에서, 웹 서비스 이용자가 직접 정보를 공유할 수 있는 블로그를 생성하는 방식(웹 2.0)이 된 것이다. 웹 2.0은 ‘참여, 공유, 개방’에 중점을 두는데, 세 키워드 모두 웹 서비스 이용자의 플랫폼 활용도를 높인 것이 그 특징이다.
